# Linux Cheat Sheet

Setup scripts and one-liners for Ubuntu, Fedora/RHEL, and WSL2 environments.

> **Versions:** Ubuntu 22.04+ · Fedora 38+ / RHEL 9+ · WSL2

---

## Homebrew

### Install Homebrew on Linux 🏠 Personal setup helper

```bash
#!/usr/bin/env bash
echo -en "\n" | /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)" && \
    echo 'eval "$(/home/linuxbrew/.linuxbrew/bin/brew shellenv)"' >> ~/.bashrc && \
    eval "$(/home/linuxbrew/.linuxbrew/bin/brew shellenv)" && \
    source ~/.bashrc && \
    brew install gcc
```

## Ubuntu

### Install Podman

```bash
#!/usr/bin/env bash
sudo apt-get update -qq
sudo apt-get -qq -y install podman
```

### Install PowerShell Core

```bash
#!/usr/bin/env bash
sudo apt-get update && sudo apt-get install -y wget apt-transport-https software-properties-common
wget -q "https://packages.microsoft.com/config/ubuntu/$(lsb_release -rs)/packages-microsoft-prod.deb"
sudo dpkg -i packages-microsoft-prod.deb
sudo apt-get update && sudo apt-get install -y powershell
pwsh -Command "Set-PSRepository -Name 'PSGallery' -InstallationPolicy Trusted"
sudo pwsh -Command Install-Module -Name Az -Force -AllowClobber -Scope AllUsers -Repository PSGallery
```

### Update CA certificates

```bash
#!/usr/bin/env bash
sudo cp *.crt /usr/local/share/ca-certificates/
sudo cp *.crt /etc/ssl/certs
mkdir -p ~/.local/share/ca-certificates/ && cp *.crt ~/.local/share/ca-certificates/
sudo update-ca-certificates
```

### Configure Python pip to use a custom CA bundle

```bash
pip config set global.cert ca.crt
echo 'export REQUESTS_CA_BUNDLE="ca.crt"' >> ~/.bashrc
```

### Install GitHub CLI

```bash
#!/usr/bin/env bash
type -p curl >/dev/null || sudo apt install curl -y
curl -fsSL https://cli.github.com/packages/githubcli-archive-keyring.gpg \
  | sudo dd of=/usr/share/keyrings/githubcli-archive-keyring.gpg
sudo chmod go+r /usr/share/keyrings/githubcli-archive-keyring.gpg
echo "deb [arch=$(dpkg --print-architecture) signed-by=/usr/share/keyrings/githubcli-archive-keyring.gpg] https://cli.github.com/packages stable main" \
  | sudo tee /etc/apt/sources.list.d/github-cli.list > /dev/null
sudo apt update && sudo apt install gh -y
```

### Install VSCode

```bash
#!/usr/bin/env bash
sudo apt-get install wget gpg
wget -qO- https://packages.microsoft.com/keys/microsoft.asc | gpg --dearmor > packages.microsoft.gpg
sudo install -D -o root -g root -m 644 packages.microsoft.gpg /etc/apt/keyrings/packages.microsoft.gpg
sudo sh -c 'echo "deb [arch=amd64,arm64,armhf signed-by=/etc/apt/keyrings/packages.microsoft.gpg] https://packages.microsoft.com/repos/code stable main" > /etc/apt/sources.list.d/vscode.list'
sudo apt-get install -y apt-transport-https
sudo apt-get update && sudo apt-get install code -y
```

### Install Azure Functions Core Tools

```bash
curl https://packages.microsoft.com/keys/microsoft.asc | gpg --dearmor > microsoft.gpg
```

### Install Starship prompt 🏠 Personal setup helper

```bash
#!/usr/bin/env bash
# Download DroidSansMono Nerd Font
FONT_URL=$(curl -s https://api.github.com/repos/ryanoasis/nerd-fonts/releases/latest \
  | grep -o "https://github.com/ryanoasis/nerd-fonts/releases/download/.*/DroidSansMono.zip")
curl -L "$FONT_URL" -o DroidSansMono.zip
unzip DroidSansMono.zip -d ~/.fonts && rm DroidSansMono.zip
fc-cache -fv

curl -sS https://starship.rs/install.sh | sh

mkdir -p ~/.config
cat > ~/.config/starship.toml <<'EOF'
command_timeout = 10000
"$schema" = "https://starship.rs/config-schema.json"
add_newline = true

[azure]
disabled = false
format = "on [$symbol($subscription)]($style) "
symbol = "# "
style = "blue bold"

[character]
success_symbol = "[➜](bold green)"

[package]
disabled = true
EOF

echo 'eval "$(starship init bash)"' >> ~/.bashrc
```

### Install Pyenv

```bash
#!/usr/bin/env bash
sudo apt-get install -y \
    git build-essential gcc make openssl libssl-dev zlib1g-dev libbz2-dev \
    libreadline-dev libsqlite3-dev python-is-python3 wget curl \
    libncursesw5-dev xz-utils tk-dev libxml2-dev libxmlsec1-dev \
    libffi-dev liblzma-dev python3-pip unzip zip python3-tk

git clone https://github.com/pyenv/pyenv.git ~/.pyenv
echo 'export PYENV_ROOT="$HOME/.pyenv"' >> ~/.bashrc
echo 'command -v pyenv >/dev/null || export PATH="$PYENV_ROOT/bin:$PATH"' >> ~/.bashrc
echo 'eval "$(pyenv init -)"' >> ~/.bashrc
source ~/.bashrc
pyenv install 3.12.0
pyenv global 3.12.0
```

### Install JetBrains Toolbox 🏠 Personal setup helper

```bash
#!/bin/bash
[ "$(id -u)" != "0" ] && exec sudo "$0" "$@"

sudo apt-get install -y libfuse2

URL=$(curl -s 'https://data.services.jetbrains.com/products/releases?code=TBA&latest=true&type=release' \
  | grep -Po '"linux":.*?[^\\]",' | awk -F ':' '{print $3":"$4}' | sed 's/[", ]//g')

FILE=$(basename "${URL}")
wget -cO "/tmp/${FILE}" "${URL}" --read-timeout=5 --tries=0

DIR="/opt/jetbrains-toolbox"
mkdir -p "${DIR}"
tar -xzf "/tmp/${FILE}" -C "${DIR}" --strip-components=1
chmod -R +rwx "${DIR}"
ln -sf "${DIR}/jetbrains-toolbox" /usr/local/bin/jetbrains-toolbox
rm "/tmp/${FILE}"
echo "JetBrains Toolbox installed."
```

## Fedora / RHEL

### Update CA certificates

```bash
#!/usr/bin/env bash
sudo dnf install -y ca-certificates
sudo cp *.crt /etc/pki/ca-trust/source/anchors/
mkdir -p ~/.pki/nssdb/ && cp *.crt ~/.pki/nssdb/
update-ca-trust
```

```bash
#!/usr/bin/env bash

    zlib-devel bzip2-devel ncurses-devel libffi-devel sqlite-devel \
    make gcc readline-devel python3-pip python3-tkinter xz-devel tk-devel git

echo 'command -v pyenv >/dev/null || export PATH="$PYENV_ROOT/bin:$PATH"' >> ~/.bashrc
```

## WSL2

### Create symlinks to Windows directories 🏠 Personal config

```bash
ln -s /mnt/c/Users          winusers
ln -s /mnt/c/Users/craig    winhome
ln -s /mnt/c/Users/craig/Desktop windesktop
```

### Run Ubuntu shell from Windows (or as root)

```bash
wsl.exe -d Ubuntu bash
wsl.exe -u root -d Ubuntu bash
```

### Set WSL2 resource limits

Run this in PowerShell - creates `~/.wslconfig` capping memory and CPU.

```powershell
@"
[wsl2]
memory=8GB
processors=2
"@ | Set-Content -Path "$Env:USERPROFILE\.wslconfig"
```

### Install WSL2 VPN toolkit (for corporate VPN compatibility)

```powershell
Invoke-WebRequest -Uri "https://github.com/sakai135/wsl-vpnkit/releases/latest/download/wsl-vpnkit.tar.gz" -OutFile "wsl-vpnkit.tar.gz"
wsl --import wsl-vpnkit "$env:USERPROFILE\wsl-vpnkit" wsl-vpnkit.tar.gz --version 2
wsl -d wsl-vpnkit
wsl.exe -d wsl-vpnkit --cd /app service wsl-vpnkit start

## Anti-patterns

- **`curl | bash` as root without reviewing the script** - executes arbitrary code from the network at root privilege. Always inspect the script first, download to a temp file, or use a checksum-verified release asset.
- **Not pinning tool versions in setup scripts** - `tfenv install latest` or `goenv install latest` will install whatever is current at runtime; a later run may break your environment. Pin to a specific version for reproducibility.
- **Sourcing `~/.bashrc` inside a non-interactive script** - `~/.bashrc` is designed for interactive shells; scripts that source it inherit aliases, `PS1`, and interactive-only settings that can cause unpredictable behaviour. Use explicit paths or pass env vars directly.
- **`sudo pip install` system-wide** - corrupts the system Python managed by apt/dnf and fights with OS package managers. Use `pipx` for CLI tools, `uv venv` or `python -m venv` for project environments.
- **Modifying `/etc/profile.d/` for user-specific config** - that directory applies to all users on the system. Put personal `PATH` and env var changes in `~/.bashrc` or `~/.profile`.
- **Hardcoding absolute paths from one user's home in shared scripts** - `/home/craig/...` paths break for any other user. Use `$HOME`, `~`, or `$(pwd)` instead.
```
